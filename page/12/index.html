06_Bean 的作用域

发表于 2020-07-31 分类于 Java 学习笔记 ， Spring
本文字数： 1.1k 阅读时长 ≈ 1 分钟
范围 描述
singleton （默认）为每个Spring IoC 容器将单个 bean 定义的作用域限定为单个对象实例
prototype 将单个 bean 定义的作用域限定为任意数量的对象实例
request 将单个 bean 定义的范围限定为单个HTTP请求的生命周期。也就是说，每个HTTP请求都有一个在单个 bean 定义后面创建的 bean 实例
session 将单个 bean 定义的范围限定为 HTTP Session的生命周期
application 将单个 bean 定义的作用域限定为ServletContext的生命周期
websocket 将单个 bean 定义的作用域限定为WebSocket的生命周期
