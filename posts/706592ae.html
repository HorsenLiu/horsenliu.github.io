07_软件安装

发表于 2020-11-23 分类于 Linux 学习笔记 阅读次数：
本文字数： 3.2k 阅读时长 ≈ 3 分钟

0. 前言

在 Linux 上安装软件一般有三种方式

  • 通过 rpm 安装包（以下用安装 JDK 来演示）
  • 通过解压安装包（以下用安装 Tomcat 来演示）
  • 通过 yum 安装（以下用安装 Docker 来演示）

1. 安装 JDK

  1. 下载好 JDK（注意版本），然后传到虚拟机中

    https://www.oracle.com/java/technologies/javase-downloads.html

  2. 执行java -version先看看有没有自带的 JDK

  3. 有的话就卸载，先找到具体的名字，再根据名字卸载，最后检查一下是否卸载干净

  4. 然后安装下载好的 JDK

  5. 配置环境变量（这里有个小问题，先不要做，下面会讲）

这里讲一下问题。根据弹幕所说，以 rpm 方式安装的 JDK 不需要配置环境变量。可以看第 4 步最后，安装好之后直接就可以通过命令看到版本号，说明此时环境变量已经有了。

然后我查了一下原因，看到了刘遄老师网站中的一篇文章：

为什么JDK的RPM包安装方式不需要配置Java环境变量
https://www.linuxprobe.com/jdk-rpm.html

问题解决了，第 5 步是不需要的。当然如果你使用压缩包解压的方式安装，还是需要这一步的。

2. 安装 Tomcat

  1. 下载好 Tomcat（注意版本），然后传到虚拟机中

    https://tomcat.apache.org/download-90.cgi

  2. 将文件移动到 /usr/tomcat/ 下，并解压

  3. 进入 bin 目录，启动 tomcat

  4. 开启 80 和 8080 端口，顺便介绍一下防火墙相关的命令

  5. 另外说一下，可以直接购买一个属于自己的域名，解析后如果端口是 80-http 或 443-https 可以直接访问，如果是 9000 8080，就需要通过 Apache 或者 Nginx 做一下反向代理。
