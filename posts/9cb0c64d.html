02_第一个 Spring Boot 程序

发表于 2020-08-17 分类于 Java 学习笔记 ， Spring Boot 基础 阅读次数：
本文字数： 3.1k 阅读时长 ≈ 3 分钟

1. 创建项目

1.1 方式一：通过网页

  1. 打开 https://start.spring.io/
  2. 填写项目信息
  3. 点击【Generate Project】按钮生成项目；下载此项目
  4. 解压项目包，用 IDEA 以 Maven 项目导入，一路下一步即可，直到项目导入完毕

1.2 方式二：IDEA 直接创建

  1. 新建项目，选择 Spring Initalizr

  2. 填写项目信息

  3. 选择依赖

1.3 项目结构分析

通过上面步骤完成了基础项目的创建，会自动生成以下文件：

  1. 程序的主启动类 HelloApplication.java
  2. 一个 application.properties 配置文件
  3. 一个测试类
  4. 一个 pom.xml

1.4 卡在Reading……pom.xml的解决方法

如果创建项目之后，进度条一直显示

Reading maven project 或者 Reading……pom.xml

  1. 检查 hosts 文件中是否有127.0.0.1 localhost且 127.0.0.1 不指向其他地址
  2. cmd 输入netsh winsock reset回车，重启电脑

2. pom.xml 分析

3. 测试运行

  1. 在 xxxApplication.java 的同级目录下建包

  2. 编写一个 Controller

  3. 运行，访问localhost:8080/hello

4. 将项目打成 jar 包

如果遇到错误，可以配置打包时跳过项目运行测试用例

成功后生成一个 jar 包，可以通过java -jar xxx.jar运行

5. 修改端口号

在 application.properties 中添加server.port=8081

6. 自定义 banner

生成 ascii 图案：https://www.bootschool.net/ascii

在 resourvces 目录下新建 banner.txt，将图案拷贝进去，重启服务。
