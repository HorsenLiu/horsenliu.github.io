12_Element-ui

发表于 2020-08-16 分类于 前端学习笔记 ， Vue 阅读次数：
本文字数： 4.6k 阅读时长 ≈ 4 分钟

1. 简介

Element-ui 是饿了么前端出品的基于 Vue.js的 后台组件库，方便程序员进行页面快速布局和构建

官网：http://element-cn.eleme.io/#/zh-CN

2. 入门

顺便捋一下新建项目的步骤

  1. 新建一个项目vue create hello-element-ui

  2. npm 安装依赖

  3. 删掉不需要的文件和代码

  4. 新建文件夹 router，views ，static

    注意：

    • static 放不会变动的文件

      static 目录下的文件并不会被 Webpack 处理：它们会直接被复制到最终的打包目录（默认是dist/static）下。必须使用绝对路径引用这些文件，这是通过在 config.js 文件中的 build.assetsPublicPath 和 build.assetsSubDirectory 连接来确定的。

    • assets 放可能会变动的文件

      assets目录中的文件会被webpack处理解析为模块依赖，只支持相对路径形式。例如，在 <img src="./logo.png">和 background: url(./logo.png)中，./logo.png是相对的资源路径，将由Webpack解析为模块依赖。
      因为 logo.png 不是 JavaScript，当被视为模块依赖时，需要使用 url-loader 和 file-loader处理它。vue-cli 的 webpack 脚手架已经配置了这些 loader，因此可以使用相对路径引用。

  5. 新建 Main.vue，Login.vue

  6. 配置路由 index.js

  7. 配置 main.js

  8. 修改 App.vue

  9. 运行npm run serve
