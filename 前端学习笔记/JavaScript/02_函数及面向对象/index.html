2. 调用函数

参数问题：JavaScript 可以传递任意个参数，也可以不传参数。所以如果参数不存在，如何避免出现问题？

3. arguments

是一个 JavaScript 免费赠送的关键词，代表传进函数的所有参数，是一个数组

问题：arguments包含所有的参数，如果想使用多余的参数进行附加操作，需要排除已有的参数。

解决方法：使用rest

4. 变量的作用域

  1. var 在函数体中声明，则无法再函数体外使用
  2. 内部函数可以访问外部函数的成员，反之不可以
  3. 假如，内部函数变量和外部函数变量重名，则内部函数会屏蔽外部函数的变量
  4. 所有变量在一开始就声明好，不要写着写着需要了，就在中途声明

5. 规范

为了避免所有的全局变量绑定到window上，导致引用不同的 js 文件，相同的全局变量发生冲突，必须设定唯一全局变量。
