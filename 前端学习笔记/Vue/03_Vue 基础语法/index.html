03_Vue 基础语法

1. v-bind

除了使用插值表达式{{}}进行数据渲染，也可以使用v-bind指令，它的简写的形式就是一个冒号:

该指令的意思是：将这个元素节点的title属性和 Vue 实例的message的值保持一致

2. v-if、v-else、v-else-if

3. v-for

在控制台输入vm.items.push({message: '123123'})会增加一条数据

4. v-on

用于将 Vue 中定义的事件，绑定到前端页面的事件上。

例如，绑定“sayHi”事件到按钮的click方法上，在点击按钮触发click事件时，对应地去做“sayHi”中的事情。

5. v-model

Vue.js 是一个 MVVM 框架，即数据双向绑定，即当数据发生变化的时候，视图也就发生变化，当视图发生变化的时候，数据也会跟着同步变化。这也算是 Vue.js 的精髓之处了。

值得注意的是，我们所说的数据双向绑定，一定是对于 UI 控件来说的，非 UI 控件不会涉及到数据双向绑定。对于我们处理表单，Vue.js 的双向数据绑定用起来就特别舒服了。

我们可以用v-model指令在表单< input>、< textarea>及< select>元元素上创建双向数据绑定。它会根据控件类型自动选取正确的方法来更新元素。尽管有些神奇，但v-model本质上不过是语法糖。它负责监听用户的输入事件以更新数据，并对一些极端场景进行一些特殊处理。
